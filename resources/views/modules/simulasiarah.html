Kasus 1: Topologi Jaringan (Tanpa Arah)

Misi: Cek jalur cadangan! Kabel LAN bersifat dua arah. Coba perhatikan, jika koneksi langsung ke Router Utama putus, apakah PC Client masih bisa dapat internet lewat temannya?

💡 Poin Belajar: Dalam graf tak berarah, sisi (garis) adalah jembatan dua arah. Derajat simpul adalah jumlah kabel yang nyolok ke PC tersebut.
# Representasi dua jenis graf undirected = { 'A': ['B', 'C'], 'B': ['A', 'C'], 'C': ['A', 'B'] } directed = { 'A': ['B'], 'B': ['C'], 'C': [] } print("Graf Tak Berarah:") for v, n in undirected.items(): print(v, "↔", n) print("\nGraf Berarah:") for v, n in directed.items(): print(v, "→", n)
